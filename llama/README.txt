1. Установить пакет `flask-cors`. Можно вручную командой в терминале: pip install flask-cors, или в олдскул стиле из бат-файла install_dependencies.bat, а также установить: pip install langchain openai
2. Разместить файл llama.config с адресом, моделью и ключем последовательно в трех строках на два уровня выше
3. Запустить app.py (можно бат-файлом run_app.bat, без консльного окна)

Это облегченная поисково-отладочная версия чат-бота. Для разработки интрефейса и настройки модели. Не для продакшена
